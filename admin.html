📊 Google Sheets 자동 저장 설정
문의가 오면 구글 스프레드시트에 자동으로 기록됩니다.
설정 방법 (1회만 하면 됩니다)
1
구글 스프레드시트 만들기
sheets.google.com 접속 → 새 스프레드시트 생성 → 이름: "홍담 문의 관리"
2
Apps Script 열기
스프레드시트 상단 메뉴 → 확장 프로그램 → Apps Script 클릭
3
아래 코드 붙여넣기
기존 내용 모두 지우고 아래 코드 붙여넣기 → 저장(Ctrl+S)
function doPost(e) { const data = JSON.parse(e.postData.contents); const sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName(data['문의유형']) || SpreadsheetApp.getActiveSpreadsheet().insertSheet(data['문의유형']); // 헤더 없으면 추가 if (sheet.getLastRow() === 0) { sheet.appendRow(['접수일시','이름','연락처','문의유형','내용']); } sheet.appendRow([ new Date().toLocaleString('ko-KR'), data['이름'], data['연락처'], data['문의유형'], data['내용'] ]); return ContentService .createTextOutput(JSON.stringify({result:'ok'})) .setMimeType(ContentService.MimeType.JSON); }
4
웹 앱으로 배포
상단 배포 → 새 배포 → 유형: 웹 앱 → 액세스: 모든 사용자 → 배포 → URL 복사
5
URL을 홍담 사이트에 입력
복사한 URL을 클로드에게 알려주시면 index.html에 바로 연동해드립니다.
✉️ 자동 답변 이메일 설정
Formspree에서 자동 답변 이메일을 설정할 수 있습니다.
Formspree 자동 답변 설정
1
Formspree 로그인
formspree.io 로그인 → 홍담문의 폼 선택
2
Autoresponse 탭 클릭
폼 설정 → Autoresponse 탭 → Enable 켜기
3
자동답변 내용 설정
Subject: 홍담오가피진 문의 접수 확인
아래 자동답변 탭의 템플릿을 복사해서 붙여넣으세요.
4
