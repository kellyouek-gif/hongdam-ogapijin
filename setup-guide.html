웹 앱으로 배포하기

1
배포 메뉴 열기

2
배포 유형 선택
유형 선택 톱니바퀴 → 웹 앱 선택
3
액세스 권한 설정
· 설명: 홍담문의자동화
· 다음 사용자로 실행: 나 (kellyouek@gmail.com)
· 액세스 권한: 모든 사용자
4
배포 & URL 복사
배포 버튼 클릭 → Google 계정 권한 허용 → 웹 앱 URL 복사
⚠️ 권한 허용 팝업이 뜨면 → "고급" 클릭 → "홍담문의자동화(으)로 이동" 클릭 → "허용"
